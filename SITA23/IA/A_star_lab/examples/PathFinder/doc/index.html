PathFinder

David Gianazza
Ecole Nationale de l'Aviation Civile
Institut de Recherche en Informatique de Toulouse

Presentation

This program implements an A* algorithm applied to a path-finding problem, using the A_star Ocaml library. The aim consists in finding the shortest path between origin and destination while avoiding obstacles.

Installation

Compile using command : 

> make

This command creates an executable file findpath and an HTML documentation that can be found in subdirectory doc/

Usage

Run the program : 

> ./findpath

Run the test program in "slow motion" : 

> ./findpath -s
